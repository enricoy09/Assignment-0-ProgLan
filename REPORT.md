# Assignment Hello World

### Tugas Utama
Berikut merupakan kode untuk program hello "Nama". Program tersebut akan meminta input nama dan akan mengeluarkan output "hello (nama)". Program ini menggunakan 2 fungsi dari std namespace, yaitu cin dan cout.

### Tugas Tambahan
Untuk mengubah file output dari myapp menjadi MyApp yaitu dengan mengubah APPNAME di makefile. Selain itu dapat juga secara manual di compile dengan "g++ -o MyApp main.cpp"
